# PeroPero.py
[Englith](./README.md) / 日本語

**PeroPero.py**は**非公式**で[**MuseDash**](https://musedash.peropero.net/)の楽曲一覧とランキングを簡単に取得するためのライブラリです。  
内部的にお問い合わせページのチーター報告用のAPIを利用しています。
